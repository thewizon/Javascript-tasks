# 💡 Bulb Toggle UI

A simple and interactive web project that simulates a real-life light bulb using HTML, CSS, and JavaScript.

## 🚀 Features

* Toggle bulb ON/OFF using a switch
* Smooth brightness transition
* Ambient light effect for realistic feel
* Keyboard control (Press **B**)
* Sound feedback on toggle

## 🛠️ Tech Used

* HTML
* CSS (Flexbox, transitions, gradients)
* JavaScript (DOM manipulation)

## 📂 Project Structure

* `bulb.html` → main file
* `off.png` → bulb image
* `sound.mp3` → toggle sound

## ▶️ How to Run

1. Open the project in VS Code
2. Click **Go Live** (Live Server)
3. Or open `bulb.html` in browser
